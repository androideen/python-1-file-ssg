Welcome to Python 1-File SSG

A lightweight, 1-file Python-based static site generator designed for simplicity and flexibility.

It transforms plain HTML files with frontmatter and simple templating tags into a fully static website. Whether you're building a personal blog or a technical documentation site, this tool provides the essential features without the bloat.

Simple Templating

Use native HTML-like tags (<template variable="..."> and <template include="...">) for includes and variables. No complex syntax to learn.

Layout Inheritance

Define global layouts once and wrap your content files easily. Perfect for maintaining a consistent design across your site.

YAML Frontmatter

Define metadata like titles, dates, and custom variables directly at the top of your HTML files using standard YAML syntax.

Asset Management

All your CSS, JS, and images in the assets/ folder are automatically handled and copied to the build directory.

Quick Start Requirements

Python 3.6+ is all you need. For advanced frontmatter and watch mode, install these optional packages:

pip install PyYAML watchdog
